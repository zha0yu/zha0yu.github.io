    文件结构:

    一个小程序主体由三个文件组成
文件类型 是否必须 作用
app.js 是 小程序逻辑
app.json 是 小程序公共设置
App.wxss 否 小程序公共样式

一个小程序页面由四个文件组成:

文件结构 是否必须 作用
wxml 是 页面结构
wxss 否 页面样式
json 否 页面配置
js 是 页面逻辑

注意：为了方便减少配置项，我们规定描述页面的这四个文件必须具有相同的路径与文件名。
